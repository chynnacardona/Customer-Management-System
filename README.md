# Customer Management System (Group Project in Information Management 2)

## Goal and Objectives of the CMS
The primary goal of the Hope, Inc. CMS is to provide a unified, secure, and immutable platform for managing customer relationships and historical business data. The system is engineered to eliminate data loss and unauthorized modifications by shifting security logic from the application layer directly to the database.
